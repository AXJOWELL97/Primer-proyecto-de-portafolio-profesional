## 🎯 Descripción del Proyecto
Este es mi primer proyecto de portafolio profesional. Se trata de una herramienta interactiva para analizar campañas de marketing bancario. El objetivo es descubrir por qué la tasa de éxito bajó del 12% al 8% y proponer soluciones basadas en datos.

La aplicación utiliza **Programación Orientada a Objetos (POO)** y ofrece un análisis exploratorio completo (EDA) con 10 ítems técnicos.
